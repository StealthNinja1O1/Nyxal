# Nyxal

Nyxal is a bot framework for AI agents on discord. It offers a full dashboard, can run multiple bots, and is very lightweight running on Preact, Vite, Bun, ElysiaJS

## Deployment

Nyxal ships as a single self-contained binary or as a Docker image. 

Quick start:

```bash
bun install
bun run dev        # dev: server :3000 + vite :5173
bun run compile    # production: standalone binary
docker compose up -d --build   # or run it in docker
```
